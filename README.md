# Player Behavior Analytics

**Player Behavior Analytics** is a Python-based desktop application created as part of an engineering thesis.  
The tool is designed to analyze, visualize, and interpret player behavior data collected from computer games and digital distribution platforms.

The application combines classic data analysis techniques with interactive visualizations and AI-assisted interpretation of results.

---

## Academic Context

This project was developed as an **engineering thesis** in the field of **Computer Science**.  
Its main goal is to demonstrate the design and implementation of a desktop analytical system that supports the exploration of player activity, preferences, and behavior patterns.
